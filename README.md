# Actua Systems - Сайт компании

Современный сайт для компании Actua Systems, производящей промышленные актуаторы на сервоприводах 750 Вт для гоночных симуляторов.

## Структура проекта

- `index.html` - главная страница
- `styles.css` - стили сайта
- `script.js` - JavaScript для интерактивности
- `README.md` - документация

## Особенности

- ✅ Современный дизайн в стиле Qubic System
- ✅ Темная тема
- ✅ Адаптивный дизайн (работает на всех устройствах)
- ✅ Плавные анимации и переходы
- ✅ Секции продуктов (актуаторы, активные ремни, кокпиты)
- ✅ Контактная форма
- ✅ Информация о компании

## Как открыть сайт

1. Просто откройте файл `index.html` в браузере
2. Или используйте локальный сервер:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - Затем откройте `http://localhost:8000`

## Настройка

### Изменение контактной информации

Откройте `index.html` и найдите секцию контактов (строка ~200). Измените:
- Email
- Телефон
- Адрес

### Добавление изображений

Замените placeholder'ы изображений на реальные фотографии ваших продуктов:
- Найдите `<div class="placeholder-image">` в HTML
- Замените на `<img src="путь/к/изображению.jpg" alt="Описание">`

### Изменение цветов

Откройте `styles.css` и измените переменные в секции `:root` (строки 7-13):
- `--bg-dark` - основной фон
- `--text-primary` - основной текст
- `--accent` - акцентный цвет

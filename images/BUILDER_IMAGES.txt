ИНСТРУКЦИЯ ПО ИЗОБРАЖЕНИЯМ ДЛЯ КОНСТРУКТОРА КОКПИТА:

Для работы конструктора кокпита поместите следующие изображения в папку images/:

ОБЯЗАТЕЛЬНЫЕ:
1. cockpit-base.jpg - базовое изображение кокпита (по умолчанию)

АКТУАТОРЫ:
2. cockpit-4dof.jpg - кокпит с актуаторами 4DOF
3. cockpit-6dof.jpg - кокпит с актуаторами 6DOF

ДИЗАЙН КОКПИТА:
4. cockpit-design-1.jpg - первый вариант дизайна кокпита
5. cockpit-design-2.jpg - второй вариант дизайна кокпита
6. cockpit-design-3.jpg - третий вариант дизайна кокпита

Рекомендуемые параметры изображений:
- Формат: JPG или PNG
- Размер: минимум 1920x1080 пикселей (16:9)
- Ориентация: горизонтальная (ландшафт)

Если изображения не найдены, будет использован placeholder.

ПРИМЕЧАНИЕ: 
- Изображения должны показывать кокпит с соответствующими компонентами
- Для дизайна можно использовать разные варианты оформления кокпита

# Как открыть сайт Actua Systems

## Вариант 1: Локально на компьютере (самый простой)

### Способ 1: Просто открыть файл
1. Найдите файл `index.html` в папке проекта
2. Дважды кликните на него
3. Сайт откроется в браузере

### Способ 2: Через браузер
1. Откройте браузер (Chrome, Firefox, Edge)
2. Нажмите `Ctrl + O` (или File → Open)
3. Выберите файл `index.html`
4. Нажмите "Открыть"

## Вариант 2: Разместить в интернете через GitHub Pages (БЕСПЛАТНО)

### Шаг 1: Включить GitHub Pages
1. Зайдите на ваш репозиторий на GitHub
2. Нажмите на вкладку **Settings** (Настройки)
3. В левом меню найдите **Pages**
4. В разделе **Source** выберите:
   - Branch: `main`
   - Folder: `/ (root)`
5. Нажмите **Save**

### Шаг 2: Дождаться публикации
- GitHub обработает сайт (обычно 1-2 минуты)
- Появится зелёная галочка и ссылка на ваш сайт

### Шаг 3: Открыть сайт
- Ваш сайт будет доступен по адресу:
  `https://YOUR_USERNAME.github.io/REPO_NAME/`
- Например: `https://fargg.github.io/actua-systems-website/`

## Вариант 3: Использовать локальный сервер (для разработки)

### Через Python (если установлен):
```bash
# В Git Bash или терминале в папке проекта
python -m http.server 8000
```
Затем откройте: `http://localhost:8000`

### Через Node.js (если установлен):
```bash
npx http-server
```
Затем откройте указанный адрес (обычно `http://localhost:8080`)

## Вариант 4: Другие бесплатные хостинги

### Netlify:
1. Зайдите на https://www.netlify.com
2. Зарегистрируйтесь
3. Подключите GitHub репозиторий
4. Сайт автоматически задеплоится

### Vercel:
1. Зайдите на https://vercel.com
2. Зарегистрируйтесь
3. Импортируйте GitHub репозиторий
4. Сайт будет доступен сразу

## Рекомендация

Для начала просто откройте `index.html` двойным кликом - это самый простой способ!

Для публикации в интернете используйте **GitHub Pages** - это бесплатно и очень просто.

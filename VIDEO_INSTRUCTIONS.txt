ИНСТРУКЦИЯ ПО ДОБАВЛЕНИЮ ВИДЕО:

1. ОТКРОЙТЕ ФАЙЛ: index.html

2. НАЙДИТЕ СЕКЦИЮ "Videos Section" (примерно строка 158)

3. ДЛЯ ДОБАВЛЕНИЯ ВИДЕО С YOUTUBE:
   - Скопируйте ID вашего видео из YouTube (например, из ссылки https://www.youtube.com/watch?v=dQw4w9WgXcQ 
     ID будет: dQw4w9WgXcQ)
   - Найдите строку: src="https://www.youtube.com/embed/YOUR_VIDEO_ID_1"
   - Замените YOUR_VIDEO_ID_1 на ID вашего видео
   - Измените название и описание видео

4. ПРИМЕР:
   Было:  src="https://www.youtube.com/embed/YOUR_VIDEO_ID_1"
   Стало: src="https://www.youtube.com/embed/dQw4w9WgXcQ"

5. ДЛЯ ДОБАВЛЕНИЯ БОЛЬШЕ ВИДЕО:
   - Скопируйте весь блок <div class="video-item">...</div>
   - Вставьте его перед закрывающим тегом </div> в секции videos-grid
   - Замените YOUR_VIDEO_ID на ID нового видео

6. ДЛЯ УДАЛЕНИЯ ВИДЕО:
   - Просто удалите весь блок <div class="video-item">...</div>

7. ДЛЯ ИЗМЕНЕНИЯ НАЗВАНИЙ И ОПИСАНИЙ:
   - Найдите <h3>Название видео 1</h3> и измените текст
   - Найдите <p>Описание работы системы</p> и измените текст

ПРИМЕЧАНИЕ: Видео автоматически адаптируются под размер экрана и сохраняют пропорции 16:9.

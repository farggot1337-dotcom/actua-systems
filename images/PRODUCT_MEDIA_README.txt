ИНСТРУКЦИЯ ПО ФОТО/ВИДЕО ДЛЯ ОКНА "УЗНАТЬ БОЛЬШЕ"

1) ФОТО (5 шт на каждый продукт)
Положите файлы в папку images/ с такими именами:

АКТУАТОРЫ:
- actuators-1.jpg
- actuators-2.jpg
...
- actuators-5.jpg

АКТИВНЫЕ РЕМНИ:
- belts-1.jpg
- belts-2.jpg
...
- belts-5.jpg

КОКПИТЫ:
- cockpits-1.jpg
- cockpits-2.jpg
...
- cockpits-5.jpg

Если какого-то файла нет — сайт покажет заглушку, но верстка не сломается.

2) ВИДЕО (по 3 шт на каждый продукт)
Откройте файл: script.js
Найдите блок PRODUCT_MEDIA и замените значения youtubeId:

Пример:
{ title: 'GT3', youtubeId: 'dQw4w9WgXcQ' }

Вставлять нужно ТОЛЬКО ID видео из ссылки:
https://www.youtube.com/watch?v=ID_ВОТ_ЭТО

3) Что должно получиться
- Кнопка "УЗНАТЬ БОЛЬШЕ" (в карточке или в секции продукта) открывает окно
- В окне: 5 фото + 3 видео (Формула 1 / GT3 / Ралли)
